# my-cpp
这是我写的一个关于通讯录管理的程序，刚刚开始学习，望指教。
This is a program I wrote about address book management, just started to learn, hope to teach.


基本功能：
  1、添加联系人
  2、显示联系人
  3、删除联系人
  4、查找联系人
  5、修改联系人
  6、清空联系人
  0、退出通讯录  
Basic functions:
  1. Add contacts
  2. Display contacts
  3. Delete a contact
  4. Search for contacts
  5. Modify the contact
  6. Clear contacts
  0. Exit the address book
